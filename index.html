Petfinder for WordPress

A collection of functions to help you display Petfinder listings on your WordPress site.

What's Included?

Petfinder for WordPress contains a handful of functions to help you quickly integrate Petfinder into your site:

  • An API feed to pull in your Petfinder data.
  • Conversions. Change the default Petfinder values into descriptive, human-readable text (ex. "Small" instead of "S").
  • A photo fetcher. Control the number and size of the photos, and display them easily with a simple function call.
  • Pet name cleanup. Remove weird characters from pet names.
  • Pet description cleanup. Remove the formatting and empty tags that get added by WYSIWYG editors.
  • A value condensor. Convert multi-word strings into single-word values that can be used as classes (useful for filtering pets with JavaScript).
  • Lists. Dynamically generate lists of all available animal types, breeds, sizes, ages, genders, and options.
  • Pet info. Grab key info, photos, descriptions and a link to their Petfinder profile for each pet in your shelter.
  • A shortcode. Display all of the info you want to share with a simple shortcode.

Note: Petfinder for WordPress is NOT a plugin. A working knowledge of PHP and WordPress functions is required. If you're looking for a plug-and-play solution, check out the Petfinder Listings plugin.

Getting Started

Getting started with Petfinder for WordPress is a simple three-step process:

  1. Add petfinder-api-for-wp.php to your theme, and call it in your functions.php file. Alternatively, copy-and-paste its contents into the file instead.
  2. Under shelter info, add your API key, shelter ID, and the number of pets you'd like to request information for.
  3. Use the [petfinder_list] shortcode in the WordPress editor to display content. Alternatively, you can add <?php echo get_petfinder_list(); ?> to any WordPress template.

Customizing Your Content

The Petfinder for WordPress are building blocks you can use to create a customized display of your available pets.

  • Adjust default Petfinder values under the get_pet_*() Conversion functions.
  • Adjust the quantity and size of photos under the get_pet_photos() function.
  • Change list outputs under the get_*_list() functions.
  • Control what gets displayed for each pet under the get_pet_info() function.

Customizing the Shortcode

The get_petfinder_list() checks to make sure that the API feed has no errors, and that their are pets to display. Then it compiles the data from your functions into an output that can be displayed with a shortcode.

You can customize what gets displayed by adding or removing functions and content from the $petfinder_list variable.

Performance Considerations

The Petfinder API can be slow, especially if you're pulling information on a large number of animals.

For better performance, try using the Quick Cache plugin. It will pre-build the page with the Petfinder API (and all other pages on your site) once an hour, significantly increasing site performance.

Increasing Timeout Limits

If you're pulling information on a large number of animals, your server may timeout before the request is completed.

If you're seeing the "Petfinder is down for the moment" message, try typing your API URL (the http://api.petfinder.com/shelter... part with all of your information added in) directly into a browser tab. If the data loads, then your server may be timing out before the request can be completed.

To fix this, increase the max_execution_time for your server. The process for doing this varies by hosting provider. You can find general information in the PHP manual.
